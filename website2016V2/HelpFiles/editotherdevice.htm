Edit Other Device

This page allows you to change or delete any device that falls into the category of "other device"

As you can see this page is virtually identical to the Add Other Device page. This page allows you to select the device you wish to edit via the drop down menu and edit any of the settings.

By clicking on [Save] you will save any changes to the set up that you have made and by clicking [Delete] you will delete the selected device from your setup.
