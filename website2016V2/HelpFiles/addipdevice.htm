Add IP Device

One of IP Mon's many advantages is that it is capable of communicating with IP (Internet Protocol) based devices, this page helps you set up these IP Devices for use with your setup.

Type : The type of IP Device you are wishing to setup

IP Address : The IP Address of the IP Device

Port : The port that you have defined for the IP Device

Image Normal : This is where you select the image to show that a device is working

By clicking on the [SAVE] button you will save the details for the specific IP device.
