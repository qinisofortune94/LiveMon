Add Other Device

This option is available for devices that may not be IP Based, normally this would be used for something like an AMF panel or a UPS unit. Most commonly these devices will work off a serial connection and this page provides for provision that.

You have the following options to choose from :

Type : This is where you select the type of device you wish to set up

IP Address : This is where you set up the IP Address for the device

Port : This is where you set up the port for the device

Serial Port: This is where you select a serial port for the device

Baud Rate : This is where you set up the baud rate of the serial port

Data Bits : This is where you set up the data bits of the serial port

Stop Bits : This is where you set up the stop bits of the serial port

Error Check : This is where you select the type of Error Checking

Handshaking : This is where you select the type of communication of the serial port.
