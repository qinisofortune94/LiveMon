Server Config

This section of IP Mon allows you to configure various aspects of your setup. Upon clicking on the server config link. You will presented with a screen shown below

.

SMS Config allows you to select what device should send an SMS Alert and in what order that specific device has priority. As in the above graphic you will see that IP Mon will send from a GSMModem before it will send from an SMTP device.

Email Config allows you to specify the mail server you wish to send from, it also allows you to specify in which order you wish IP Mon to send from.

General allows you to specify the SMS Timer and Email Timer, this value determines how long after an alert is generated, that you recieve notification via SMS or Email.

To schedule reports, simply name the report, how long a period you wish to monitor and then on what days you wish to receive the report.
