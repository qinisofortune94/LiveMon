Edit IP Device

Occasionally you will want to edit or remove and IP device, this page assists you with this. Upon clicking on the Edit IP Device link you will be shown the following drop down menu.

Once you have selected a device, you will see the screen below.

As you can see, this screen is virtually identical to the Add IP Device page.

You can click on [Save] to save your changes or you can click [Delete] to remove the device all together.
