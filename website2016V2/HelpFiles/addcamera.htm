Add Cameras

IP Mon's main feature is the ability to use IP based cameras, this page allows you to setup a new camera for your system with the least amount of trouble. There are a variety of options that can be set here, for example the reasons why a camera would start to record an image. You can read more on this topic with reference to installing cameras elsewhere in this help document.

 IP Address : This is where you setup the IP Address the camera uses

Port : This is where you choose the port the camera will use

Username : This is where you enter the username for the camera

Password : This is where you enter the password for the above mentioned username

Caption : This is where you enter the name of camera. ie. Server Room Camera

Type :

PreEvent Recording : This is where you select the PreEvent Recording Time

PostEvent Recording : This is where you select the PostEvent Recording Time

Recording Events :You have multiple options here, these options define what will cause the camera to start recording
