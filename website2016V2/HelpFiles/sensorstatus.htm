Sensor Status

The sensor status screen displays the status of all the sensors at any given time. As you can see, your sensors are displayed with the captions that you gave them, as well as displaying the corresponding images you selected for each possible status.

This screen can also be displayed in the banner bar, by clicking on Status, this allows you to view the status of all the sensors while you work elsewhere in IP Mon.

If you click on Logo, it will bring back the original IP Mon logo.
